# منصة قراءة القرآن - v0.1 Prototype

هذه أول نسخة تجريبية منظمة من الصفحة.

## ما الذي تم عمله؟

- فصل الصفحة إلى ملفات منظمة:
  - `styles/` للتصميم
  - `src/` للمنطق البرمجي
  - `data/` لبيانات القرآن والترجمة والتفسير لاحقًا
  - `assets/` للخطوط والأيقونات
  - `vendor/` للمكتبات المحلية لاحقًا
- إضافة نظام نوافذ موحد `ModalManager`.
- إضافة دعم زر الرجوع في الهاتف للنوافذ.
- جعل الضغط على الآية يفتح قائمة إجراءات.
- إضافة ملاحظات نصية.
- إضافة رسم بالقلم على Canvas.
- حفظ الملاحظات كـ JSON.
- حفظ الرسم كصورة PNG أو JSON.
- تصدير آية كصورة باستخدام html2canvas مؤقتًا من CDN.
- إضافة PWA مبدئي `manifest.webmanifest` و `sw.js`.
- الصوت Online فقط ولا يتم تخزينه.

## تشغيل النسخة

ضع ملف بيانات القرآن باسم:

```text
quran.json
```

بجوار `index.html`، أو داخل:

```text
data/quran.json
```

ثم شغّل المشروع من سيرفر محلي، مثل:

```bash
python -m http.server 8080
```

ثم افتح:

```text
http://localhost:8080
```

## ملاحظات مهمة

- لا تفتح `index.html` مباشرة من File Explorer لأن `fetch` و Service Worker غالبًا لن يعملوا بشكل صحيح.
- مكتبة `html2canvas` مستخدمة من CDN مؤقتًا. عند تثبيت التطبيق النهائي، ننقلها إلى `vendor/`.
- الخطوط يجب وضعها داخل:

```text
assets/fonts/
```

بالأسماء:

```text
AmiriQuran.ttf
Kitab-Regular.woff2
```
