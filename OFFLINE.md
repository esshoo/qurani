# تشغيل التطبيق بدون إنترنت

## ما الذي يعمل بدون إنترنت؟

- واجهة التطبيق.
- ملفات CSS و JavaScript.
- الخطوط والأيقونات إذا كانت موجودة داخل `assets/`.
- ملف القرآن إذا كان موجودًا في `data/quran.json` أو `quran.json` وتم فتح التطبيق مرة واحدة على الأقل.
- الملاحظات والرسومات والعلامات والصور المحفوظة داخل IndexedDB.
- التفسير المحلي إذا وضعته داخل `data/tafsir/` بنفس بنية التطبيق.

## ما الذي لا يعمل بدون إنترنت؟

- الصوت، لأنه يظل Online فقط افتراضيًا.
- التفسير الخارجي من الإنترنت إذا لم يكن محفوظًا محليًا.

## مكان ملف القرآن المفضل

ضع الملف هنا:

```text
data/quran.json
```

أو بجوار `index.html`:

```text
quran.json
```

## ملاحظة مهمة

لازم تشغيل التطبيق من سيرفر محلي أو HTTPS حتى يعمل Service Worker.
مثال:

```bash
python -m http.server 8000
```
