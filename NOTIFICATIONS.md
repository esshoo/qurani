# إشعارات الورد - v0.5

## ما الذي يعمل الآن؟

- طلب صلاحية الإشعارات من الإعدادات.
- تفعيل تذكير يومي بوقت محدد.
- تخصيص عنوان ونص الإشعار.
- إرسال إشعار تجريبي.
- فتح التطبيق عند الضغط على الإشعار.

## حدود مهمة

إشعارات الويب المحلية لا تضمن العمل إذا كان المتصفح أو التطبيق مغلقًا تمامًا على كل الأنظمة.
للتشغيل الموثوق حتى والتطبيق مغلق، نحتاج لاحقًا إلى:

1. Service Worker.
2. Push API subscription.
3. خادم يرسل Web Push في الوقت المحدد.
4. قاعدة بيانات تحفظ اشتراكات المستخدمين.

الصوت لا يتم تخزينه ولا يتم تشغيله داخل الإشعار.
